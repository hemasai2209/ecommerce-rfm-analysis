# 🛒 E-Commerce Sales & Customer Insights Dashboard

End-to-end data analytics project using real-world e-commerce transaction data to uncover revenue trends, top-performing products, and customer behavior patterns through RFM segmentation.

## 📊 Project Overview
- Analyzed **96,478 delivered orders** generating **R$15.42M** in revenue
- Built an RFM (Recency, Frequency, Monetary) model to segment customers into 6 behavioral groups
- Designed an interactive Power BI dashboard with KPI cards, slicers, and drill-down charts

## 🎯 Objective
Help an e-commerce business identify its most valuable customers, understand revenue drivers, and design targeted retention strategies.

## 🛠️ Tools & Technologies
- **Python** (Pandas, NumPy, Matplotlib, Seaborn) — data cleaning, EDA, RFM modeling
- **Power BI** — interactive dashboard design
- **Excel** — intermediate data storage

## 📂 Dataset
[Olist Brazilian E-Commerce Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) (Kaggle) — 100k+ real orders, 2016–2018

## 🔍 Key Insights
- Revenue grew from near-zero in 2016 to **R$8.5M by 2018**
- São Paulo (SP) state alone generated **R$5.8M**, far ahead of other regions
- **Champions** (21.67% of customers) drove a disproportionate share of total revenue
- **24.82%** of customer value is classified as "Lost" — a major re-engagement opportunity
- Top product category: **Beleza & Saúde (Health & Beauty)** at R$1.41M

## 📈 Customer Segments (RFM)
| Segment | Description |
|---|---|
| Champion | Recent, frequent, high spenders |
| Loyal Customer | Consistent repeat buyers |
| New Customer | Recent first-time buyers |
| Potential Loyalist | Engaged, room to grow |
| At Risk | Used to be active, going quiet |
| Lost | Long inactive, low value |
